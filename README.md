# PYTHON - DJANOG Full Stack Guide

## Installation

- Install python 3.7
- Install pip latest

## Django

Install environments manager

`$ pip3 install pipenv`

Create environments

`$ sudo -H pip install -U pipenv`

Run dev server

`$ python manage.py runserver [port]`

Run migrations

`$ python manage.py migrate`

Create a new app

`$ python manage.py startapp polls`
